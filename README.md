# HF Personal Byte Vault

Simple HackForums Byte Vault that can be hosted locally or on a server.

Minimum Deposit & Withdraw is 100 Bytes.

----
Create a new API App:
```
https://hackforums.net/usercp.php?action=apinew
```

Get client_id and secret_key from: 
```
https://hackforums.net/usercp.php?action=apideveloper
```

----
**Edit config.php with your client_id and secret_key**
